Вопрос 2: Что такое git rebase?Для чего он нужен?Разница между git rebase и git merge?
Ответ: git rebase это каманда которая позваляет переместить одну ветку на другую в отличие от merge Rebase изменяет историю коммитов

Вопрос 3: Что такое git fetch?Разница между git fetch и git pull?
Ответ:git fetch это команда которая скачивает изменения из удалённого репозитория на настоящий .Git pull загружает коммиты из удаллённого репозитория
